# java_labs
Discipline: "Big Data programming languages"

## Run code
1. Start docker container:
```bash
docker compose run --rm app bash
```
2. Compile project
```bash
javac path/to/lab/Main.java
```
3. Run code
```bash
java path.to.lab.Main
```
